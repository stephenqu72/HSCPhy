To solve this problem, we must treat the motion of the ball as a projectile. The key principle is that **horizontal motion** and **vertical motion** are independent of each other.

*   **Horizontal velocity ($v_x$):** $3.0\text{ m s}^{-1}$ (constant, as there is no air resistance).
*   **Vertical acceleration ($g$):** $10\text{ m s}^{-2}$ downwards.
*   **Time of flight ($t$):** $0.40\text{ s}$.

---

### a. Calculate the horizontal distance
The horizontal distance ($\Delta x$) is found using the formula:
$$\Delta x = v_x \times t$$
$$\Delta x = 3.0 \times 0.40 = 1.2\text{ m}$$

---

### b. Calculate the height of the table
The height ($h$) corresponds to the vertical displacement ($\Delta y$). Since the initial vertical velocity ($u_y$) is $0$, we use the kinematic equation:
$$\Delta y = u_y t + \frac{1}{2}gt^2$$
$$\Delta y = 0 + \frac{1}{2} \times 10 \times (0.40)^2$$
$$\Delta y = 5 \times 0.16 = 0.80\text{ m}$$
The height of the table is **$0.80\text{ m}$**.

---

### c. Calculate the speed at which the ball hits the floor
The final speed ($v$) is the magnitude of the resultant velocity vector, which has a horizontal component ($v_x$) and a vertical component ($v_y$).

1.  **Horizontal velocity:** $v_x = 3.0\text{ m s}^{-1}$
2.  **Vertical velocity:** $v_y = u_y + gt = 0 + (10 \times 0.40) = 4.0\text{ m s}^{-1}$
3.  **Resultant speed:** Using Pythagoras' theorem:
    $$v = \sqrt{v_x^2 + v_y^2}$$
    $$v = \sqrt{3.0^2 + 4.0^2}$$
    $$v = \sqrt{9 + 16} = \sqrt{25} = 5.0\text{ m s}^{-1}$$

The speed at which the ball hits the floor is **$5.0\text{ m s}^{-1}$**.

To solve this problem, we need to understand the relationship between linear speed ($v$), angular velocity ($\omega$), and the radius of the circular path ($r$).

### Step-by-Step Explanation:

1.  **Identify the variables:**
    *   For object A: Radius = $2r$, linear speed = $v_1$.
    *   For object B: Radius = $r$, linear speed = $v_2$.
    *   Given: Both objects have the same angular velocity, $\omega_A = \omega_B = \omega$.

2.  **Recall the physics formula:**
    The relationship between linear velocity ($v$) and angular velocity ($\omega$) for an object moving in a circle of radius $r$ is:
    $$v = r\omega$$

3.  **Apply the formula to objects A and B:**
    *   For object A: $v_1 = (2r)\omega$
    *   For object B: $v_2 = (r)\omega$

4.  **Calculate the ratio:**
    We are asked for the ratio of $v_1$ to $v_2$:
    $$\frac{v_1}{v_2} = \frac{2r\omega}{r\omega}$$
    
    By cancelling out common terms ($r$ and $\omega$):
    $$\frac{v_1}{v_2} = \frac{2}{1}$$

The ratio of $v_1$ to $v_2$ is $2:1$. Looking at the provided options, this corresponds to option **(A)**.

### Conclusion:
The linear velocity is directly proportional to the radius when the angular velocity is constant. Since object A is twice as far from the centre as object B, its linear speed must be twice as high.

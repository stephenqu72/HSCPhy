To solve this problem, we need to use the kinematic equations of motion. Since the ball is thrown vertically downwards, we can define the downward direction as positive.

### Given Data:
*   Initial velocity ($u$): $7.5\text{ ms}^{-1}$
*   Time ($t$): $3.5\text{ s}$
*   Acceleration due to gravity ($a$): $g \approx 9.8\text{ ms}^{-2}$
*   Displacement ($s$): This represents the height of the building ($h$)

### Formula:
We use the displacement equation:
$$s = ut + \frac{1}{2}at^2$$

### Calculation:
Substitute the known values into the equation:
$$h = (7.5 \times 3.5) + \frac{1}{2} \times 9.8 \times (3.5)^2$$

1. Calculate the first term:
   $$7.5 \times 3.5 = 26.25$$

2. Calculate the second term:
   $$\frac{1}{2} \times 9.8 \times 12.25 = 4.9 \times 12.25 = 60.025$$

3. Add the two results:
   $$h = 26.25 + 60.025 = 86.275\text{ m}$$

### Sample Answer

**1. Aim:** To determine the horizontal launch speed ($v_x$) of a ball bearing.

**2. Equipment:** Projectile launcher, ball bearing, meter ruler, carbon paper/white paper, plumb line, and a stopwatch (optional depending on method, but we will use the horizontal displacement method).

**3. Method:**
1. Secure the launcher to a table edge and ensure it is perfectly horizontal. Use a plumb line to mark the point on the floor directly beneath the release point of the launcher. Let this be $(0,0)$.
2. Measure the vertical height ($h$) from the release point of the ball to the floor using a meter ruler.
3. Fire the ball bearing from the launcher. Use carbon paper on the floor to mark the landing spot.
4. Measure the horizontal distance ($d$) from the plumb line point to the landing spot.
5. Repeat the launch 3–5 times to calculate an average value for $d$ to improve reliability.

**4. Calculations:**
Because the projectile has an initial vertical velocity of zero ($u_y = 0$), we can treat the vertical and horizontal motions independently.

*   **Step 1: Calculate time of flight ($t$).**
    Using the vertical displacement equation:
    $$h = u_y t + \frac{1}{2}gt^2$$
    Since $u_y = 0$:
    $$h = \frac{1}{2}gt^2 \implies t = \sqrt{\frac{2h}{g}}$$
    *(Where $g \approx 9.8 \text{ m/s}^2$)*

*   **Step 2: Calculate launch speed ($v_x$).**
    Since there is no acceleration in the horizontal direction, $v_x$ is constant:
    $$v_x = \frac{d}{t}$$
    Substitute the expression for $t$:
    $$v_x = \frac{d}{\sqrt{\frac{2h}{g}}}$$

**5. Reliability:** Perform multiple trials and average the range ($d$) to reduce the impact of random errors.

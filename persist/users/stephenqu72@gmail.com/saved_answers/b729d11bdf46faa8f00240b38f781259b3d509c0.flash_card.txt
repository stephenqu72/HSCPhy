### Front
How does a stationary object's path appear to an observer moving at a constant velocity?

### Back
- Key idea: Relative Motion (Galilean transformation)
- Formula / law / constant / figure: $y = \frac{g}{2v^2}x^2$ (Parabolic path)
- When to use it: To determine the trajectory of an object relative to a non-inertial or moving reference frame.
- Common trap: Assuming the object follows the same path for all observers; the path shape depends entirely on the relative velocity between the observer and the object.

To answer this question, we must apply the scientific definitions of **accuracy** and **precision** in the context of experimental measurement:

*   **Accuracy:** Refers to how close a measured value is to the true or target value (the bullseye).
*   **Precision:** Refers to how close a set of measurements are to each other (i.e., how tightly clustered the results are).

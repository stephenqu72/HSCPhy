### Question 17 Analysis
To find the current in the primary circuit, we use the principle of conservation of energy for an *ideal* transformer, which states that the input power equals the output power: $P_p = P_s$.

1.  **Calculate Secondary Power ($P_s$):**
    Using the voltage $V_s = 240\text{ V}$ and resistance $R = 1200\text{ }\Omega$:
    $$P_s = \frac{V_s^2}{R} = \frac{240^2}{1200} = \frac{57600}{1200} = 48\text{ W}$$
2.  **Calculate Primary Current ($I_p$):**
    Since the transformer is ideal, $P_p = P_s = 48\text{ W}$.
    The input power is also defined by $P_p = V_p \times I_p$.
    Given $V_p = 40\text{ V}$:
    $$48 = 40 \times I_p$$
    $$I_p = \frac{48}{40} = 1.2\text{ A}$$

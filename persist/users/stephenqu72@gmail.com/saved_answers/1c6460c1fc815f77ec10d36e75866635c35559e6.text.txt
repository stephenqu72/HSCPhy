To solve this problem, we need to analyze how each variable in the range formula affects the final value of $R$. The given formula is:

$$R = \frac{u^2 \sin(2\theta)}{g}$$

We want to find which change results in a new range $R' = 2R$. Let’s evaluate the options provided:

*   **(A) $u$ is doubled:** If we replace $u$ with $2u$, the new range $R'$ becomes:
    $$R' = \frac{(2u)^2 \sin(2\theta)}{g} = \frac{4u^2 \sin(2\theta)}{g} = 4R$$
    This increases the range by a factor of 4, not 2.
*   **(B) $g$ is doubled:** If we replace $g$ with $2g$, the new range $R'$ becomes:
    $$R' = \frac{u^2 \sin(2\theta)}{2g} = \frac{1}{2}R$$
    This halves the range.
*   **(C) $\theta$ is changed from $30^\circ$ to $45^\circ$:** 
    For $30^\circ$: $R = \frac{u^2 \sin(60^\circ)}{g} = \frac{u^2 \frac{\sqrt{3}}{2}}{g}$
    For $45^\circ$: $R' = \frac{u^2 \sin(90^\circ)}{g} = \frac{u^2}{g}$
    The ratio $R'/R = \frac{1}{\sqrt{3}/2} = \frac{2}{\sqrt{3}} \approx 1.15$. This does not result in doubling the range.
*   **(D) $u$ is multiplied by $\sqrt{2}$:** If we replace $u$ with $u\sqrt{2}$, the new range $R'$ becomes:
    $$R' = \frac{(u\sqrt{2})^2 \sin(2\theta)}{g} = \frac{2u^2 \sin(2\theta)}{g} = 2R$$

Since multiplying $u$ by $\sqrt{2}$ results in $2R$, option **(D)** is the correct answer.

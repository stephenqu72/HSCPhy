To calculate the initial velocity of the projectile, we need to find both its horizontal ($v_x$) and vertical ($v_y$) components.

### 1. Determining the horizontal velocity ($v_x$)
The range $R$ is given as $130\text{ m}$. Looking at the graph, the projectile returns to its original vertical displacement at $t = 6\text{ s}$. This is the total time of flight $T$.
Since horizontal velocity is constant:
$$v_x = \frac{\text{Range}}{\text{Time of flight}} = \frac{130\text{ m}}{6\text{ s}} \approx 21.67\text{ m/s}$$

### 2. Determining the initial vertical velocity ($v_y$)
We can use the kinematic equation for vertical displacement:
$$y = v_{y}t + \frac{1}{2}a_y t^2$$
Using the point on the graph at $t = 6\text{ s}$, where vertical displacement $y = 0$ and acceleration $a_y = -9.8\text{ m/s}^2$:
$$0 = v_y(6) + \frac{1}{2}(-9.8)(6)^2$$
$$0 = 6v_y - 176.4$$
$$6v_y = 176.4 \implies v_y = 29.4\text{ m/s}$$

*(Alternatively, note that the peak is at $t = 3\text{ s}$. At the peak, vertical velocity is $0$. Using $v = u + at$: $0 = v_y + (-9.8)(3) \implies v_y = 29.4\text{ m/s}$.)*

### 3. Calculating the resultant initial velocity ($v$)
The magnitude of the initial velocity is given by:
$$v = \sqrt{v_x^2 + v_y^2}$$
$$v = \sqrt{(21.67)^2 + (29.4)^2}$$
$$v = \sqrt{469.59 + 864.36} = \sqrt{1333.95} \approx 36.52\text{ m/s}$$

To find the angle $\theta$:
$$\theta = \tan^{-1}\left(\frac{v_y}{v_x}\right) = \tan^{-1}\left(\frac{29.4}{21.67}\right) \approx 53.6^\circ$$

**Final Answer:** The initial velocity is approximately **$36.5\text{ m/s}$** at an angle of **$53.6^\circ$** above the horizontal.

### How to solve this:
1. **Analyze the Motion:** Recognize that projectile motion is separated into independent horizontal and vertical components.
2. **Find Horizontal Velocity:** Use the provided range and the total flight time from the graph ($R = v_x \times T$).
3. **Find Vertical Velocity:** Use the time of flight or time to peak ($v_y = u_y + at$) to find the initial vertical component.
4. **Vector Combination:** Use the Pythagorean theorem to combine the perpendicular velocity components into a single resultant vector.

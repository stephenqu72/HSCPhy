To solve this projectile motion problem, we assume standard gravity $g = 9.8 \, \text{m s}^{-2}$ and ignore air resistance.

### (a) Calculating the horizontal distance $d$

First, we determine the time $t$ it takes for the package to hit the ground using the vertical displacement formula:
$$y = u_y t + \frac{1}{2} g t^2$$
Since the initial vertical velocity $u_y = 0$, the equation simplifies to:
$$250 = 0 + \frac{1}{2}(9.8)t^2$$
$$250 = 4.9t^2$$
$$t^2 = \frac{250}{4.9} \approx 51.02$$
$$t \approx 7.143 \, \text{s}$$

Now, we calculate the horizontal distance $d$ covered during this time, knowing the horizontal velocity $v_x$ remains constant at $52.5 \, \text{m s}^{-1}$:
$$d = v_x \times t$$
$$d = 52.5 \times 7.143 \approx 375 \, \text{m}$$

**The package should be released at a distance of approximately $375 \, \text{m}$ from the hikers.**

---

### (b) Calculating the speed of the package as it hits the ground

The speed $v$ is the magnitude of the resultant velocity vector: $v = \sqrt{v_x^2 + v_y^2}$.

1.  **Horizontal velocity ($v_x$):** Remains constant at $52.5 \, \text{m s}^{-1}$.
2.  **Vertical velocity ($v_y$):** Calculated using $v_y = u_y + gt$:
    $$v_y = 0 + (9.8)(7.143) \approx 70.0 \, \text{m s}^{-1}$$

Now, find the resultant speed:
$$v = \sqrt{(52.5)^2 + (70.0)^2}$$
$$v = \sqrt{2756.25 + 4900}$$
$$v = \sqrt{7656.25} \approx 87.5 \, \text{m s}^{-1}$$

**The speed of the package as it hits the ground is approximately $87.5 \, \text{m s}^{-1}$.**

---

### How to solve this type of problem
1.  **Separate dimensions:** Always treat horizontal (constant velocity) and vertical (constant acceleration due to gravity) components independently.
2.  **Use time as the bridge:** The time taken to fall vertically is exactly the same time the object travels horizontally. Solve for $t$ using the vertical motion first.
3.  **Vector Addition:** When asked for final "speed," remember to combine the horizontal and vertical velocity components using the Pythagorean theorem, as speed is a scalar.

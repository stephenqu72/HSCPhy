### Question 7
**Answer:** (B) Decrease the vertical sensitivity.

**Explanation:**
The "Vertical sensitivity" dial controls how many volts are represented by each vertical square (division) on the screen. 
- If the waveform is too small (less than one division), we need to zoom in vertically.
- "Decreasing the sensitivity" (e.g., changing the setting from $2\text{ V/div}$ to $0.5\text{ V/div}$) makes each square represent a smaller voltage value. As a result, the same input voltage will cause the electron beam to deflect over more vertical divisions, effectively increasing the visual size of the amplitude on the screen.

### Question 8
**Answer:** (C) Increase the time base.

**Explanation:**
The "Time base" dial controls the horizontal scale, representing how much time each horizontal square covers (e.g., $10\text{ ms/div}$). 
- If the student can only see one period, it means the horizontal scale is currently too "zoomed in." 
- To fit more cycles (periods) onto the screen, you need each horizontal division to represent a *longer* duration of time. By "increasing the time base" (e.g., moving from $20\text{ ms/div}$ to $50\text{ ms/div}$), you compress the waveform horizontally, allowing multiple periods to fit within the width of the display.

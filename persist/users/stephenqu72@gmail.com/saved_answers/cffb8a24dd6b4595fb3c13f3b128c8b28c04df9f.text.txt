### Explanation
To solve this problem, you must understand the principle of emission spectroscopy. Every element has a unique "fingerprint" represented by its emission spectrum—a specific set of lines at distinct wavelengths (or positions on the spectrum).

If a sample contains a specific element, **every single line** present in that element's reference spectrum must appear in the spectrum of the sample. 

1.  **Analyze the reference spectrum:** Looking at the top diagram, note the positions of the lines. There are four distinct lines spread across the spectrum.
2.  **Evaluate Sample A:** This sample does not contain the same number of lines or the same relative spacing as the reference.
3.  **Evaluate Sample B:** Observe that the four lines from the reference spectrum are clearly present within the pattern of Sample B. While Sample B contains additional lines (which indicates it is a mixture containing other elements as well), the presence of *all* the original reference lines confirms that this sample contains the element in question.
4.  **Evaluate Sample C and D:** These samples lack the specific arrangement of lines found in the reference spectrum, meaning they do not contain the element.

In analytical chemistry, we look for the "fingerprint" of the target element. Since the pattern of the reference spectrum is a subset of the pattern in Sample B, we conclude that B contains the element.

To solve this problem, we start with the formula for centripetal force:

$$F_c = \frac{mv^2}{r}$$

### Step-by-Step Explanation:

1. **Identify the variables for the car:**
   - Mass $m_c = 1000\text{ kg}$
   - Velocity $v_c = v$
   - Force $F_c = \frac{1000v^2}{r}$

2. **Identify the variables for the truck:**
   - Mass $m_t = 3000\text{ kg}$
   - Let the required velocity be $v_t$
   - The radius $r$ remains the same for the same corner.

3. **Set the forces equal to each other:**
   Since the problem states the centripetal force must be the same:
   $$\frac{m_c v_c^2}{r} = \frac{m_t v_t^2}{r}$$

4. **Simplify and solve for $v_t$:**
   - We can cancel $r$ from both sides:
     $$m_c v_c^2 = m_t v_t^2$$
   - Substitute the known values:
     $$1000 \cdot v^2 = 3000 \cdot v_t^2$$
   - Divide both sides by 3000:
     $$v_t^2 = \frac{1000}{3000}v^2$$
     $$v_t^2 = \frac{1}{3}v^2$$
   - Take the square root of both sides:
     $$v_t = \sqrt{\frac{1}{3}}v$$
     $$v_t = \frac{v}{\sqrt{3}}$$

Comparing this result to the given options, the correct answer is **(A)**.

To solve this problem, we must first analyze the information provided. The question asks for the current in the secondary coil, but it provides information about the resistance of the primary coil. 

### Step-by-Step Explanation:

**1. Analyze the "Ideal Transformer" Assumption**
In an ideal transformer, we assume there is no flux leakage, no eddy current losses, and, most importantly for this specific question, **zero resistance in the coils**. When a problem states a transformer is "ideal," the resistance values provided for the coils are typically "distractors"—information intended to test if you understand the theoretical model. In an ideal transformer, the voltage transformation is determined solely by the turns ratio:
$$\frac{V_p}{V_s} = \frac{N_p}{N_s}$$

**2. Evaluate the given information**
*   Primary turns ($N_p$) = 100
*   Secondary turns ($N_s$) = 1000
*   Primary voltage ($V_p$) = 230 V
*   Primary resistance ($R_p$) = 15 $\Omega$ (Note: This is irrelevant for an ideal transformer model).

**3. Address the missing information**
To find the secondary current ($I_s$), we typically use the relationship $P_p = P_s$, or $V_p I_p = V_s I_s$. However, we are not given the load resistance on the secondary side or the primary current ($I_p$). 

Looking at the options provided, there is a common ambiguity in these types of physics problems. If we calculate the primary current as if the transformer were not there (using Ohm's Law $I = \frac{V}{R}$):
$$I_p = \frac{230\text{ V}}{15\text{ }\Omega} = 15.33\text{ A}$$
If we then apply the ideal transformer relationship for current:
$$\frac{I_p}{I_s} = \frac{N_s}{N_p} \implies I_s = I_p \times \frac{N_p}{N_s}$$
$$I_s = 15.33\text{ A} \times \frac{100}{1000} = 1.533\text{ A}$$

Rounding this to the closest value provided in the options, we get **1.5 A**.

**Conclusion:**
By treating the primary resistance as the determinant for primary current and then using the turns ratio to find the secondary current, we arrive at option (A).

**Correct Option: (A) 1.5 A**

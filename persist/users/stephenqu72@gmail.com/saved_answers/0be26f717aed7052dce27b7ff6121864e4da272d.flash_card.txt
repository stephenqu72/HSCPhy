### Front
What is the structural framework for answering an HSC Physics "Working Scientifically" (experimental design) question?

### Back
- **Key idea:** Structure response using four components: Method, Accuracy (systematic error reduction), Reliability (repetition/outliers), and Safety (hazard/risk/control).
- **Formula / law / constant / figure:** No fixed constant or figure.
- **When to use it:** For any 5–9 mark "Design an investigation" or "Evaluate an experiment" question.
- **Common trap:** Failing to explicitly link the improvement (e.g., repeating trials) to the specific experimental goal (e.g., increasing reliability). Mentioning "accuracy" but failing to describe *how* systematic errors were reduced (e.g., calibration).

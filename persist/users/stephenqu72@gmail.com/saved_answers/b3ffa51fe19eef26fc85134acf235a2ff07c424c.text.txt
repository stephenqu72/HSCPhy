To solve this problem, we need to understand the nature of $\beta^-$ (beta-minus) decay.

### Step-by-Step Explanation:

1.  **Analyze the initial nucleus:**
    The isotope is Nickel-65, written as ${}_{28}^{65}\text{Ni}$.
    *   The atomic number ($Z$) is 28, representing the number of protons.
    *   The mass number ($A$) is 65, representing the sum of protons and neutrons.
    *   The number of neutrons ($N$) is $A - Z = 65 - 28 = 37$.

2.  **Understand $\beta^-$ decay:**
    In $\beta^-$ decay, a neutron in the nucleus converts into a proton, an electron (the $\beta^-$ particle), and an anti-neutrino.
    *   The proton number increases by 1 ($Z \rightarrow Z + 1$).
    *   The neutron number decreases by 1 ($N \rightarrow N - 1$).
    *   The total mass number ($A$) remains constant because the decrease in neutrons is balanced by the increase in protons ($65 - 1 + 1 = 65$).

3.  **Determine the daughter nucleus properties:**
    *   New number of protons: $28 + 1 = 29$.
    *   New number of neutrons: $37 - 1 = 36$.
    *   (Check: $29 + 36 = 65$, which matches the original mass number).

4.  **Match with the options:**
    The daughter nucleus has 29 protons and 36 neutrons. This corresponds to option **A**.

### Answer:
The correct answer is **A**.

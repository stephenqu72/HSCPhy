### Answer and Explanation

**(a) Sketch graph of $h$ vs $x$**
In projectile motion without air resistance, the path follows a parabolic trajectory. Since the horizontal displacement $x$ is directly proportional to time ($x = v_x t$), the relationship between height and horizontal distance remains a parabola. You should draw an inverted U-shape starting at $(0,0)$ and ending on the $x$-axis at some distance $R$.

**(b) Expression for initial vertical component**
Using basic trigonometry on the velocity vector triangle:
$$V_y = u \sin \theta$$

**(c) Deduce $\theta = 30^\circ$**
To solve this, we use the kinematics formula for vertical motion: $v_y = u_y + a_y t$.
1.  **Identify values from the graph:** The bullet reaches its maximum height at $t = 10\text{ s}$. At maximum height, the vertical velocity $v_y = 0$.
2.  **Use acceleration:** We know $a_y = -g \approx -9.8\text{ m/s}^2$ (standard gravity).
3.  **Calculate $u_y$:** 
    $$0 = u_y + (-9.8 \times 10)$$
    $$u_y = 98\text{ m/s}$$
4.  **Use the result from (b):**
    $$98 = 200 \sin \theta$$
    $$\sin \theta = \frac{98}{200} = 0.49$$
    *Note:* Given that $g$ is often approximated as $10\text{ m/s}^2$ in some HSC contexts, let's check: If $g = 10\text{ m/s}^2$, then $u_y = 100\text{ m/s}$.
    $$100 = 200 \sin \theta$$
    $$\sin \theta = 0.5$$
    $$\theta = \arcsin(0.5) = 30^\circ$$
The data confirms $\theta = 30^\circ$ when using the standard approximation of $g = 10\text{ m/s}^2$.
